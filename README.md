# PRODIGY_ML_03

## Cats vs Dogs Image Classification using SVM

This project is part of my Machine Learning Internship at Prodigy InfoTech.

### Task Objective
Implement a Support Vector Machine (SVM) model to classify images of cats and dogs.

### Technologies Used
- Python
- OpenCV
- Scikit-learn
- Matplotlib
- NumPy

### Features
- Image preprocessing
- Image resizing and grayscale conversion
- SVM model training
- Cat vs Dog prediction
- Accuracy evaluation

### Output
The model predicts whether the given image is a cat or a dog and displays the prediction result.
